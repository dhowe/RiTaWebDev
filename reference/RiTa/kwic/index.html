Class

RiTa

Name

kwic
Description Return a list of occurrences of the key word in the Key-Word-In-Context (KWIC) model. RiTa.concordance() must called before calling kwic().
Note : this API is not available in Java yet.
Example 
// Java: API not available yet


// JavaScript

let opts = {
 ignoreCase: false,
 ignoreStopWords: true
};

RiTa.concordance(text, opts);
let lines = RiTa.kwic(word, numWords);
Parameters
stringthe word as the keyword
intupper limit of words before or after the keyword in the returned sentences. (optional, default=6)
Returns
string[]all the occurrences of the keyword in the model, each with no more than 'numWords' words of context on either side
Related tmp_related
Note See https://en.wikipedia.org/wiki/Concordance_%28publishing%29
Syntax 
RiTa.kwic(word);
RiTa.kwic(word, numWords);
Platform JavaScript
