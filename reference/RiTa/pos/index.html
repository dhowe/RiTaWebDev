Class

RiTa

Name

pos
Description Uses the default PosTagger to tag the input with tags from the PENN tag set or the simplified tag set [a, r, v, n]
Example 

RiTa.pos("I am a cat."); 
// returns -> [ "prp", "vbp", "dt", "nn", "." ]
Parameters
Stringthe text to be tagged
booleanif true, use WordNet style tags (optional, default=false)
Object
(or Map in Java)		optional way to set the option

{boolean} option.simple:
use WordNet style tag or not
Returns
String[]Array of part-of-speech tags
Related RiTa.analyze()

Syntax 
RiTa.pos(word);
RiTa.pos(word, boolean);
RiTa.pos(word, options);
