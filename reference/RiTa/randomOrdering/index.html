Class

RiTa

Name

randomOrdering
Description Returns a randomly ordered array of unique integers from 0 to numElements.
The size of the array will be numElements.
Example 
array = RiTa.randomOrdering(5);
Parameters
int
(or an array, this option not available in Java yet)		numElements
(or an array to randomly re-order, this option not available in Java yet)
Returns
int[]
(or another array if an array is used as the parameter)		unique integers from 0 to numElements-1
(or the re-ordered array if an array is used as parameter)

Note tmp_note
Syntax 
RiTa.randomOrdering(numElements);
RiTa.randomOrdering(array);//not available in Java yet
